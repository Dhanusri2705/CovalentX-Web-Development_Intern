Task 03 - Stopwatch Web Application

This project was developed as part of the CovalentX Technologies Web Development Internship.

Features
- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Record Multiple Lap Times
- Responsive Design
- Modern Glassmorphism UI
- Dynamic DOM Manipulation
- Interactive User Experience

Technologies Used
- HTML5
- CSS3
- JavaScript

Project Objective
To build an interactive stopwatch application that allows users to measure elapsed time efficiently while demonstrating front-end development skills using HTML, CSS, and JavaScript.
